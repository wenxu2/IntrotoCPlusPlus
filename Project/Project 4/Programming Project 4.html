COP 2334: C++ Programming

Programming Project # 4

 

Prerequisite reading: Chapters 6 & 7

 

 Objective:

  Our supervisor wants a ready to use program to calculate the lowest, highest and average salary an employee earned in ‘N’ number of months. This program reads employee salary data from a file.  The program outputs the average, lowest, and highest salary from the file.   

 

Outcomes:

1.      Read data from an input file and push it into an array.

2.      Process the data in an array with functions.

3.      Have a function accept an array as a parameter.

4.      Use for and while loops when processing an array.

 

Specifications:

1.      You must use an array to store the salary data (the entries should stored as a float or double so that we can keep precision).

2.      We will only store up to 20 months of salaries (The max capacity of the array is 20).

3.      The input file contains a report that lists the quantity of salaries followed by the salaries themselves.

4.      Sample Input file format: 

 

3

3200.92

300.83

1352.17  

 

The '3' means that we need to calculate the lowest, highest and average salary the employee earned in three months. The numbers each represent the salary for one month.  

 

5.      The main() function should be modular, which means that the subtasks should be performed by functions (remember functional decomposition from the prior project). It is up to you to implement the functional decomposition, parameters and return types.

6.      The tasks are:  

a.      Prompt the user for the name of the file that has the salary data and populate the array with the data from the specified.  Remember to make sure the file exists and that it opens correctly.

b.      Average the data – please only display up to two places after the decimal.

d.      Get the highest and lowest salary in the file – again, please only display up to two places after the decimal point.  Remember to clean up after yourself and close the file ðŸ˜Š

7.      main() should output the above to the user.

8.      The user should be able to rerun the program as many times as they wish.   

9.      Sample output:

 

Enter the salary file name: proj4salary.dat

The average salary is 1617.97   

The highest salary is 3200.92

The lowest salary is 300.83
