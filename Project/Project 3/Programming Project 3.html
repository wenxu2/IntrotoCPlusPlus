Project Requirements:

We will create a converter program for data quantities.  We will give the user choices and will carry out the requested operation. The menushould appear as follows:

Welcome to the super duper wizard converter!

     1) KB --> MB

     2) KB --> GB

     3) MB --> KB

     4) MB --> GB

     5) GB --> MB

     6) GB --> KB

     7) Exit

Please enter an integer from 1 to 7 > 

Notes:

1. You should create a choice() function that is called in a loop from the main() function that displays the choices.

2. The menu from choice() will display the options as above and take user input until a valid number in the range is input.   Upon detecting valid input it will return the integer chosen.

3. When the choice() function returns proper input, code in the main() function will get the necessary input from the user and then call the function that calculates the result, and will display that result.

4. *Important* - Note that our units change by 2^10, that is, 1024.  1GB = 1024MB, 1 MB = 1024 KB, and 1KB = 1024 bytes.  1KB is NOT 1000bytes, 1MB is NOT 1000KB, and 1GB is NOT 1000MB.

5. You are to get the data quantity in terms of the original units and convert to the new units using the appropriate functions you coded.  You should then display the new quantity as a number (with decimals), properly labeled. 

6. We don't know how may times the user will want to run a conversion from the main.
